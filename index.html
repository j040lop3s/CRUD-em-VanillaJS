Lista Dinâmica | CRUD Completo

    Este projeto apresenta um sistema onde trabalho - em JavaScript Puro, HTML e CSS - as operações básicas para criar, ler, atualizar e deletar informações. Tais operações fundamentam todo o conceito tráfego de dados web através de variadas interfaces de usuário.

    O código cria uma array que armazena os itens adicionados, estes itens são exibidos em uma lista dinâmica juntamente com botões de remoção e atualização. Além dos métodos e propriedades comuns na criação e manipulação de elementos DOM, fiz uso de .parentElement() que retorna o elemento pai em função do target do evento. Utilizei métodos de array como .push() para adicionar itens, .indexOf() para localizar o índice de um item específico e .splice() para remover um item da array.
