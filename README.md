# CRUD Completo com HTML + CSS + JavaScript

Este projeto foi desenvolvido utilizando conceitos puros de Desenvolvimento Web para executar e demonstrar as 4 operações básicas envolvidas no cotidiano de qualquer sistema, em uma funcionalidade de adicionar, remover, atualizar e ler os itens
inseridos em uma lista dinâmica.

## Arquitetura e Técnicas utilizadas

A arquitetura do projeto é simples, com separação de arquivos em pastas por responsabilidade e atuação. O script executa
toda manipulação DOM , e o controle dos itens removidos é feito no array de armazenamento e com o método DOM .remove(). Mais
informações são descritas no conteúdo da página.
